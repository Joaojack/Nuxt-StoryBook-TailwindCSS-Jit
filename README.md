# Nuxt + Storybook + Tailwind Jit

**WORK IN PROGRESS**
Boilerplate config with example,  **Nuxt + Storybook + Tailwind jit**.
Story in **.mdx**
I don't know if it's the perfect way (i'm a beginner), but for what i needed it worked...

## Build Setup
```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# launch Storybook
$ npx nuxt storybook

```
